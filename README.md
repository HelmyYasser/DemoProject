## 👨‍💻 About This Project

This repository demonstrates a **scalable and maintainable UI Test Automation Framework** built using modern industry standards.

It reflects real-world QA Automation practices including:

- ✅ Page Object Model (POM)
- ✅ Clean code & reusable utilities
- ✅ Structured logging
- ✅ Advanced reporting with Allure
- ✅ Maven-based dependency management
- ✅ TestNG suite configuration
- ✅ Maintainable and extensible architecture

This project represents my hands-on expertise as a **QA Automation Engineer**.

---

## 🏗 Framework Architecture

| Layer | Responsibility |
|-------|---------------|
| **Base Layer** | Driver setup, teardown, shared configuration |
| **Page Layer** | Encapsulates UI elements & actions (POM) |
| **Test Layer** | Contains test logic & assertions |
| **Utilities Layer** | Reusable helpers (waits, logging, config) |
| **Reporting Layer** | Allure integration & failure screenshots |
| **Build Layer** | Maven dependency & lifecycle management |

---

### 🔄 Execution Flow

Test Class  
→ BaseTest Setup  
→ Driver Initialization  
→ Page Interaction  
→ Assertion (TestNG)  
→ Logging & Screenshot  
→ Allure Reporting  
→ Driver Teardown
